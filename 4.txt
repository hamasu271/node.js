Day 4: Node.js 웹 서버 구축 문제 과제

1. **Node.js의 HTTP 모듈의 주요 역할은 무엇인가요?**
    
    - b) 웹 서버 구축
   
HTTP 모듈은 서버를 생성하고 HTTP요청을 처리하는데 사용됩니다.

2. **RESTful API에서 `GET` 메서드는 주로 어떤 작업을 수행하나요?**

    - d) 리소스 조회

서버로부터 데이터를 요청하여 조회하는역할을 합니다.

3. **MVC 패턴에서 'View'의 역할은 무엇인가요?**
   
    - b) 사용자 인터페이스 구성
   
뷰는 말그대로 데이터를 눈으로 보여주는 역할을 담당합니다.

4. **URL과 URI의 차이점에 대해 옳은 설명은 무엇인가요?**
    
    - d) b와 c 모두 옳다.

모든 URL은 URI의 한 형태이며, URI는 URL과 URN을 포함하는 개념입니다.

5. **JSON Web Token (JWT)의 세 가지 주요 구성 요소는 무엇인가요?**
   
 - a) 헤더, 페이로드, 서명
   
6. **OAuth에서 'Resource Owner'는 누구를 의미하나요?**
    
    - c) 리소스에 대한 접근 권한을 가진 사용자
    Resource Owner는 자신이 소유한 자원에 대한 접근 권한을 제공하는 사용자입니다.

7. **RESTful API에서 리소스는 주로 어떤 형태로 식별되나요?**
   
    - b) URI
   
8. **JWT의 페이로드에는 어떤 정보가 포함될 수 있나요?**
  
    - c) 클레임 (예: 사용자 정보)
   

9. **OAuth의 주요 목적 중 하나는 무엇인가요?**
  
    - c) 사용자 자격 증명 보호
    
OAuth는 외부 애플리케이션에 사용자 비밀번호를 노출하지 않고 권한을 위임하는 방식입니다.

10. **MVC 패턴에서 'Controller'의 역할은 무엇인가요?**
    
    - c) 사용자 입력을 처리하고 모델과 뷰를 연결
    
 Controller는 사용자 입력을 받아 모델에 전달하고, 처리된 데이터를 View에 전달하여 화면에 표시합니다.
